# 医院在线预约挂号系统 — 项目架构方案

> Spring Boot + Vue 3 SPA + MyBatis-Plus + Vite + Pinia
> 从 Vue 2 MPA 原型 重构 为前后端分离项目

---

| 后端框架 | Spring Boot | 2.7.x / 3.x | REST API |

## 三、通用约定（全团队遵守）

| 项目 | 值 |
|------|-----|
| 后端基路径 | `http://localhost:8080/api` |
| 请求格式 | `Content-Type: application/json` |
| Token 传递 | `Authorization: Bearer <token>` |
| 统一响应 | `{ code: 200, message: "success", data: {} }` |
| 错误码 | 200/400/401/403/404/500 |
| 分页请求 | `?page=1&pageSize=10` |
| 分页响应 | `{ records: [], total: 0, page: 1, pageSize: 10 }` |
| 字段命名 | DB `snake_case` → API `camelCase` |

## 四、前端架构（Vue 3 SPA）

### 4.1 初始化命令

```bash
pnpm create vite@latest frontend -- --template vue
cd frontend
pnpm add vue-router@4 pinia axios
pnpm add -D @vitejs/plugin-vue
pnpm dev
```

### 4.2 入口文件 `src/main.js`

```js
import { createApp } from 'vue'
import { createPinia } from 'pinia'
import App from './App.vue'
import router from './router'
import './styles/global.css'

const app = createApp(App)
app.use(createPinia())
app.use(router)
app.mount('#app')
```

### 4.3 根组件 `src/App.vue`

```vue
<template>
  <router-view v-slot="{ Component }">
    <keep-alive>
      <component :is="Component" />
    </keep-alive>
  </router-view>
</template>
```

### 4.4 路由设计 `src/router/index.js`（25 条）

```
/login                          → LoginView.vue
/register                       → RegisterView.vue
/change-password                → ChangePasswordView.vue
/                               → HomeView.vue
/hospitals                      → HospitalListView.vue
/hospital/:id                   → HospitalDetailView.vue
/doctors                        → DoctorListView.vue
/doctor/:id                     → DoctorDetailView.vue
/diseases                       → DiseaseListView.vue
/disease/:id                    → DiseaseDetailView.vue
/articles                       → ArticleListView.vue
/article/:id                    → ArticleDetailView.vue
/search                         → SearchView.vue
/reservation                    → ReservationView.vue
/reservation/pay/:orderNo       → ReservationPayView.vue
/reservation/success/:orderNo   → ReservationSuccessView.vue
/consult                        → ConsultView.vue
/consult/pay/:orderNo           → ConsultPayView.vue
/consult/success/:orderNo       → ConsultSuccessView.vue
/profile                        → ProfileView.vue
/family-members                 → FamilyMemberView.vue
/my-appointments                → MyAppointmentView.vue
/my-consults                    → MyConsultView.vue
/my-reviews                     → MyReviewView.vue
/follow                         → FollowView.vue
/messages                       → MessageView.vue
/feedback                       → FeedbackView.vue
```

**路由守卫**：除 `/login` `/register` `/` `/hospitals` `/hospital/:id` `/doctors` `/doctor/:id` `/diseases` `/disease/:id` `/articles` `/article/:id` `/search` 外，其余页面需登录。

### 4.5 关键文件

| 文件 | 作用 | 位置 |
|------|------|------|
| `api/request.js` | Axios 封装：baseURL + Bearer Token + 401 拦截 + 统一错误提示 | 前端 |
| `stores/auth.js` | Pinia：login/logout/fetchUser + localStorage 持久化 | 前端 |
| `router/index.js` | 路由表 + `beforeEach` 守卫 | 前端 |
| `styles/variables.css` | CSS 变量：`--primary: #1e88e5`, `--bg: #f5f7fa` | 前端 |
| `styles/global.css` | Reset + 全局排版 | 前端 |

### 4.6 从原型到 Vue 3 的迁移对照

| 原型（Vue 2 MPA） | 新架构（Vue 3 SPA） |
|-------------------|---------------------|
| 32 个 `.html` 各独立 | 1 个 `index.html` + 25 条路由 |
| `Vue.component()` 全局注册 | `.vue` SFC + `import` |
| localStorage 分散读写 | Pinia store `useAuthStore()` |
| 每页重复 header/footer CSS | Scoped + `variables.css` |
| `window.location.href` 跳转 | `router.push()` |
| 内联 Mock JSON 数据 | Axios → Spring Boot |
| 无构建，浏览器直读 | `pnpm build` → `dist/` |

### 5.2 核心配置 `application.yml`

```yaml
server:
  port: 8080

spring:
  datasource:
    url: jdbc:mysql://localhost:3306/hospital_db?useSSL=false&serverTimezone=Asia/Shanghai&characterEncoding=utf8
    username: root
    password: 123456
    driver-class-name: com.mysql.cj.jdbc.Driver

mybatis-plus:
  configuration:
    map-underscore-to-camel-case: true    # 自动转换 snake_case → camelCase
    log-impl: org.apache.ibatis.logging.stdout.StdOutImpl
  global-config:
    db-config:
      id-type: assign_id                   # 雪花算法主键

# JWT 配置
jwt:
  secret: hospital-jwt-secret-key-2026
  expire: 86400000                         # 24小时(毫秒)
```

## 七、与原型的关键差异

| 差异点 | 原型 | 新架构 |
|--------|------|--------|
| 搜索页 | 5 个独立 HTML | 1 个 SearchView 内 Tab 切换 |
| 关注页 | 3 个独立 HTML | 1 个 FollowView 内 Tab 切换 |
| 登录态 | localStorage 分散读写 | Pinia + Axios 拦截器统一管理 |
| 路由守卫 | 无，手动跳转 | `router.beforeEach` |
| 样式 | 每个页面重复 5000+ 行 CSS | Scoped + CSS 变量 |
| 构建 | 无 | `pnpm build` → `dist/` |
| 数据 | 内联 Mock JSON | Axios → Spring Boot REST API |
